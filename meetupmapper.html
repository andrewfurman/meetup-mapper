About MeetupMapper:

This site uses the meetup.com API to find events in your area and displays the events on a Google map so that you can see where they are.

Instructions:

1. Type in the zip code that you would like to find events near.

2. Enter the number of days into the future in which you would like to find events.

3. Enter the radius of miles around the zip code in which you would like to find events.

4. Click Find Meetups!
